# Project Brief: jekyll-auto-thumbnails

## Overview
Jekyll plugin that automatically generates and serves optimized image thumbnails for faster page loads. Scans rendered HTML, generates thumbnails with intelligent caching, and seamlessly replaces img src attributes.

## Repository
- **GitHub**: https://github.com/Texarkanine/jekyll-auto-thumbnails
- **License**: AGPL-3.0-or-later
- **Ruby Version**: >= 3.4.0

## Core Dependencies
- jekyll >= 4.0, < 5.0
- nokogiri ~> 1.15

## Development Dependencies
